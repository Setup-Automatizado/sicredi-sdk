# Changelog

All notable changes to this project will be automatically documented here by [semantic-release](https://github.com/semantic-release/semantic-release).

## 1.0.1 (2025-01-09)

### Initial Release

- PIX API support (cob, cobv, pix, loc, webhook)
- Boleto Hibrido (cobv) support
- Lote CobV batch operations
- mTLS certificate authentication
- OAuth2 token management with auto-refresh
- Dual runtime support (Bun + Node.js)
- Zero runtime dependencies
- Exponential backoff retry with jitter
- Typed error hierarchy
- Standalone webhook payload parser
